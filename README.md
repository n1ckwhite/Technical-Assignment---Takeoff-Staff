# Тестовое задание Takeoff-Staff

## Проект запускать командой: npm start

## Версия node js: 17.3.1

### В проекте реализована функциональность:
- Адаптивность,
- Авторизация,
- Фильтрация карточек,
- Добавление карточки,
- Удаление карточки,
- Редактирование карточки

## Технологии проекта:
- React,
- Redux,
- Redux Усилители,
- React-Router,
- TypeScript
